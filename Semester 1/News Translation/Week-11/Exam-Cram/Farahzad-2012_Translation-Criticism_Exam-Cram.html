4. Facet 1: Intertextuality

Key Concepts

Theorist: Julia Kristeva (French narratologist)

Core Idea: Texts relate to, reference, quote each other

Every Text Has Two Parts:

  1. Referential/Repetitive: Quoted from other texts
  2. Non-referential/New: Created by author

Application to Translation

Replace: "Equivalence" ‚Üí "Intertextuality"

Why:

  • Target ‚â† copy of source
  • Different author, readers, language, society

Target Text Has:

  • Repetitive part from source (concepts, themes, forms)
  • Non-repetitive part due to:
    • Linguistic/cultural/social differences
    • Translator's choices

Example: Khayyam's Rubaiyat

  • Persian original ‚Üí FitzGerald's English ‚Üí Secondary translations (German, Russian, French, etc.)
  • All share commonalities BUT differ from each other
  • Relationship = intertextual, NOT equivalence

5. Facet 2: Critical Discourse Analysis (CDA)

Core Assumptions

Fairclough (1995): Language, power, ideology are interconnected

  • Language = vehicle of ideology
  • Text = arena of ideology
  • Word choices ‚Üí positioning ‚Üí power centering/marginalizing

Ideology in Translation Studies

Broader than political meaning

Calzada Perez (2003): "Ways of thinking, evaluating, behaving that are promoted in society, become dominant, turn into custom"

B. Grammatical Choices

1. Active/Passive:

  • "France put..." ‚Üí "...was put" - hides agent
  • If repeated ‚Üí removes agency from source society

2. Nominalization:

  • "Children play role" ‚Üí "Children's role" - changes from agent to patient

3. Positive/Negative:

  • "stopped" vs. "didn't advance" - different realities

4. Verb Tense:

  • "were excluded" ‚Üí past perfect vs. present perfect
  • Past = historical; Present = ongoing condition

5. Element Order:

  • First position = more emphasis

C. Translation Strategies

  • Deletion: If patterned ‚Üí censorship
  • Addition: If repeated ‚Üí changes representation (e.g., Gone with the Wind - female character)
  • Substitution: ŸÖÿ≠ÿ±ÿßÿ® ‚Üí "chapel" - suggests mosque = church

Analyzes: Translator/publisher notes, prefaces, afterwords, footnotes

Example 1 - Colonial Footnotes:

  • Western translations of Indian literature
  • Numerous footnotes made India seem exotic/mysterious
  • Paved way for colonialism (per Appiah)

Example 2 - FitzGerald's Preface:

  • "Persian poetry needs little artistic refinement"
  • Reveals Orientalist perspective
  • Devalues Persian/Eastern poetry

Example 3 - Coleman Barks' Rumi:

  • Doesn't know Persian, yet translates Rumi
  • Includes Hindu cooking recipe in afterword
  • Devalues the work

  • English: Burning American flag (hatred)
  • Persian: American military presence (NOT hatred)
  • Difference = ideological difference

7. Model Methodology

Process

  1. Analyze translation choices
  2. Explain ideologically with attention to social/political/historical context
  3. Conduct at all three levels: textual, paratextual, semiotic

Type: Comparative criticism

Focus: What translation DOES (not what it IS)
